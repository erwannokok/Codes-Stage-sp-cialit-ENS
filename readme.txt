Pour executer les codes, il suffit de se placer dans le dossier "code+data" et de run le code en faisant par exemple "python Interface_Camera_Neuro" si l'on veut lancer l'interface dédiée à l'interface de la caméra neuro. Pour celle-ci, il suffira d'ouvrir un fichier CSV en cliquant sur "Ouvrir". 

Pour l'interface classique "Interface_Camera_Classique" ou le spatio-temporel "Spatio-Camera_Classique" il suffit d'ouvrir le dossier de photos. ( Format TIFF )
